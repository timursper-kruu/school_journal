ЭЖ «Шкала»: история версий


1.5.446 21.07.2023 Добавлена возможность учета уроков (занятий), пропущенных
   учителем по уважительной причине (праздничный день, распоряжение
   администрации и т. п.)

1.4.440 29.03.2023 Добавлена возможность просмотра всех текущих отметок
   по всем предметам в интерфейсе обучающегося.

1.3.435 21.11.2022 Исправлена ошибка отображения списка предметов при
   редактировании педагогической нагрузки.

1.3.434 15.11.2022 Исправлена ошибка отображения уважительной причины пропуска
   уроков для пользователей, не являющихся администраторами или классными
   руководителями.

1.3.433 13.07.2022 Исправлена ошибка определения дат промежуточной аттестации
   при экспорте журналов классов в файл в июле.

1.3.429 22.06.2022 Создан сервис генерирования сводной (для всего класса)
   ведомости четвертных, полугодовых и годовых отметок.

1.2.424 29.01.2022 Для использования API сторонними сервисами добавлены
   cors-заголовки ответа сервера. При показе отметок (кроме страниц межклас-
   сных групп внеурочной деятельности) ячейки журнала, соответствующие датам,
   для которых у данного обучающегося имеется документ об уважительной причине
   пропуска уроков, показываются со светло-зеленым фоном.

1.1.421 16.10.2021 Изменены условия запроса pin-кода для выставления или
   редактирования отметки (без pin-кода теперь можно редактировать отметки
   в течение месяца после окончания учебного периода).

1.0.418 22.08.2021 Исправлен баг отображения списка предметов в интерфейсе
   главного администратора.

1.0.416 02.07.2021 Исправлен баг ограничения доступа администратора к сервису
   генерирования табеля.

1.0.410 06.05.2021 Исправлен баг потери списка подгрупп и курсов внеурочной
   деятельности при редактировании данных пользователя.

1.0.409 23.04.2021 Исправлены мелкие баги разбора URL сервером и определения
   педагогической нагрузки сервисом «Явка».

1.0.406 12.04.2021 Внесены изменения в способ разбора URL сервером.

1.0.398 03.04.2021 Добавлена функция экспорта журнала межклассной группы.

0.13.392 26.03.2021 Добавлена возможность работы с заметками для межклассных
   групп.

0.12.390 04.03.2021 Запущен базовый функционал сервиса «Внеурочная
   деятельность».

0.11.367 15.02.2021 Колонка экзаменационных отметок и соответствующая строка
   в интерфейсе учащегося показываются на более темном фоне.

0.10.361 29.08.2020 Создан сервис «Явка», показывающий статистику посещаемости
   всех учащихся (или учащихся группы классов) за один день.

0.9.357 22.05.2020 Добавлены возможности: создания межклассных групп для
   внеурочной деятельности и закрепления их за учителями; комплектования
   учителями состава этих межклассных групп.

0.8.329 26.04.2020 Добавлена возможность создания ссылок в домашнем задании.

0.7.321 11.03.2020 Добавлена функция вывода статистических данных (сведений
   об успеваемости и о количестве проведенных учителями часов).

0.6.311 02.03.2020 Создан комплект скриптов и служебных файлов для локального
   просмотра сохраненных файлов базы данных журнала за прошлые годы.

0.5.277 02.02.2020 В случае наличия новых заметок для учащегося или родителя
   ему показывается иконка-уведомление.
   
0.5.273 29.01.2020 Добавлены функции работы с заметками для обучающихся и их
   родителей и просмотра логов авторизации пользователей.

0.4.256 19.01.2020 Добавлены функции: экспорта журнала одного класса в файл;
   ввода и отображения количества часов одного занятия.

0.3.227 01.12.2019 Измененен порядок отображения и интерфейс ввода весов
   отметок учителями.

0.3.223 18.09.2019 Добавлено генерирование табелей отметок промежуточной
   аттестации.

0.2.211 18.08.2019 Изменена система загрузки клиентских скриптов.

0.2.201 14.08.2019 Добавлены учет документов о причинах пропуска уроков,
   автоматический подсчет и публикация данных о посещаемости, вывод статистики
   на предметных страницах журнала и в дневнике учащегося.

0.1.150 09.08.2019 Добавлено логирование запросов, а также перенаправление
   http-запросов на https.

0.0.143 19.07.2019 Одностраничное веб-приложение с базовым функционалом
   для серверной платформы NodeJS с клиентской частью на JavaScript.
   Используется СУБД NeDB.
